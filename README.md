# 🚀 Responsive Landing Page

A modern and responsive landing page built using **HTML, CSS, and JavaScript**. This project demonstrates responsive web design principles and an interactive navigation system that enhances user experience across different devices.

---

## 📌 Project Description

The Responsive Landing Page is designed for a technology company, **TechNova Solutions**. The website provides a clean and modern interface with a fixed navigation menu that remains visible while scrolling.

The navigation bar dynamically changes its appearance when users scroll down the page and provides hover effects on menu items for better interactivity. The layout is fully responsive, ensuring a seamless experience on desktops, tablets, and smartphones.

## ⚙️ How It Works

### Navigation Hover Effect

When the user hovers over a menu item:

* Background color changes
* Text color changes
* User interaction becomes more engaging

### Navigation Scroll Effect

When the user scrolls down:

* Navigation bar background changes
* Shadow effect is added
* Menu becomes more visible

### Responsive Layout

Using CSS media queries:

* Desktop view displays horizontal navigation
* Mobile view displays a compact menu
* Content adjusts automatically based on screen size
